# 贪知蛇-贪吃蛇

加入各种题目，玩家需要吃掉正确的含有正确答案的苹果，可以在玩经典游戏贪吃蛇时学习到知识，用户资料和排行榜可以向朋友证明你的实力！

> [!IMPORTANT]
> 需要安装Pygame，使用此命令安装
> ```
> pip3 install pygame
> ```
